Java based matrix classes for ruby

This file provides a module JBLAS which contains efficient matrix implementations based on java and BLAS/LAPACK implementations.

For double precision computations, DoubleMatrix and DoubleVector are provided, for single precision computations, FloatMatrix and FloatVector.

Basically, the classes provided are the actual java classes with syntactic sugar added (for example for arithmetic operations). This is integrates very nicely, however, not that the full list of methods is not available in rdoc.
